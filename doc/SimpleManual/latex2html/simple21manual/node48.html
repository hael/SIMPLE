Program: simple_print_fsc

simple_print_fsc is a program for printing the binary FSC files produced by PRIME3D 
    $ SIMPLE_PRINT_FSC smpd=<sampling distance(in A)> box=<image size(in pixels)>}
    fsc=<fsc_state1.bin>
