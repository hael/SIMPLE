Context variables: projects — queryset of all projects available to this user workspaces — queryset of workspaces for the selected project current_project_id — id of the currently selected project (0 = none) current_workspace_id — id of the currently selected workspace (0 = none) iframeurl — URL to load inside the content iframe, or None Selecting a project or workspace submits a GET form, reloading the page with the new selection reflected in the URL query string. {% endcomment %} {% load static %} SIMPLE Stream
