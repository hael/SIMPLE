Motion Correction and CTF Estimation
Perform alignment of movie frames to correct for movement and estimate CTF parameters for each resulting micrograph
Job Parameters
Job Name
An identifier for the job. This will appear in the name of the folder containing the output from the job. E.G First_100
Job Description
A description of the job
Required Arguments
Movie List
Text file containing a list of movies. Usually this is the produced during an import job
Sampling Distance
Sampling distance of the microscope in A/pixel
Acceleration Voltage
Acceleration voltage of the microscope (kV)
Spherical Abberation
Spherical abberation of the microscope (mm)
Amplitude Contrast
Fraction of the the amplidtude contrast used for fitting CTF
Optional Arguments
File Body
File body prepended to each output file
Start Low Pass Limit
Start low-pass limit (A). Default: 15A
Stop Low Pass Limit
Stop low-pass limit (A). Default: 8A
Maximum Halfwidth Shift
Maximimum halfwidth shift (pixels)
Exposure Time
Total exposure time (seconds)
Power Spectrum Dimensions
Power spectrum dimensions (pixels)
Number String Length
Length of number string (characters)
First Iteration
Start iterating from here
Scale Image
Image scale factor
Number Frames To Group
Number of frames to average
First Frame
Start frame index
Final Frame
Final frame index
Sigma Cutoff
Sigma cutoff for outlier rejection
Output Filename
High Pass Limit
High pass limit(A)
Low Pass Limit
Low pass limit (A)
Minimum Defocus
Minimum expected defocus (microns)
Maximim Defocus
Maximum expected defocus (microns)
Expected Astigmatism
Expected astigmatism (microns)
Astigmatism Step
Step size for astigamtism search (microns)
Phaseplate
Images obtained with phaseplate. Default: no
