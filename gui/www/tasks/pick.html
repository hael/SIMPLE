Reference Based Picking
Automatically pick particles using a set of references
Job Parameters
Job Name
An identifier for the job. This will appear in the name of the folder containing the output from the job. E.G First_100
Job Description
A description of the job
Required Arguments
Micrograph List File
Text file contaiing a list of the micrographs to pick
Sampling Distance
Sampling distance of the microscope in A/pixel
References
References to pick with
Optional Arguments
Low Pass Limit
Low pass limit (A)
Picking Threshold
threshold (binarisation: 0-1; distance filer: in pixels)
Remove Outliers
Remove outliers. Default: yes
