Extract Particles
Extract particles from micrographs
Job Parameters
Job Name
An identifier for the job. This will appear in the name of the folder containing the output from the job. E.G First_100
Job Description
A description of the job
Required Arguments
Micrograph Parameters File
Text file containing a list of micrographs and associated CTF parameters. Usually this is output from an unblur, unblur_ctf or pipeline job
Optional Arguments
Boxfiles Directory
Directory containing box files. These files must have the same name as the micrograph with a .box suffix
Boxsize
Box size in pixels if different from that defined in the box files
Invert Contrast
Invert the contrast of extract particles. Default: yes
Extract Outside
Extract boxes extending outdside the micrograph boundaries. Default: no
