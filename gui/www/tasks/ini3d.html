Initial Model Generation
Generate an initial 3D model from 2D cluster averages
Job Parameters
Job Name
An identifier for the job. This will appear in the name of the folder containing the output from the job. E.G First_100
Job Description
A description of the job
Required Arguments
Particle Stack File
Stack file containing extracted particles. Usually this is the produced during an extract job
Sampling Distance
Sampling distance of the microscope in A/pixel
Mask Radius
Mask radius (pixels)
Point Group Symmetry
Point group symmetry (cn|dn|t|o|i)
Optional Arguments
High Pass Limit
High pass limit(A)
Low Pass Limit
Low pass limit (A)
Fraction Of Clusters
Fraction of clusters to include in model. Default: 1
Envelope Masking
Enable envelope masking. Default: no
Low Pass Limit For Envelope Masking
Low pass limit for envelope mask generation (A)
Inner Mask Radius
Inner mask radius (pixels)
Inner Mask Falloff
Falloff of inner mask (pixels). Default: 10 pixels
Edge Size
Edge size for softening envelope (pixels)
Molecular Weight
Molecular weight (kD)
Binary Layers Grown For Envelope
Binary layers grown for molecular envelope(pixels))
Number Of Projection Directions
Number of projection directions
Use Shift Barrier Constraint
Use shift barrier constraint. Default: yes
