2D Clustering
Sort particles into clusters based on their similarity in 2D
Job Parameters
Job Name
An identifier for the job. This will appear in the name of the folder containing the output from the job. E.G First_100
Job Description
A description of the job
Required Arguments
Particle Stack File
Stack file containing extracted particles. Usually this is the produced during an extract job
Sampling Distance
Sampling distance of the microscope in A/pixel
Mask Radius
Mask radius (pixels)
Number Of Clusters
Number of clusters to sort the particles into
Apply CTF Correction
Choose to apply full CTF correction, perform phase flipping only or none
Optional Arguments
CTF Parameters File
File containing CTF parameters
Initial References
Initial references to start the clustering
Orientation Table File
File containing previously determined orientations for each particle
High Pass Limit
High pass limit(A)
Low Pass Limit
Low pass limit (A)
Starting Low Pass Limit
Starting low pass limit (A). Default: 15A
Stopping Low Pass Limit
Starting low pass limit (A). Default: 8A
Low Pass Limit For Centering Binarisation
Low pass limit for binarisation in centering (A)
Maximum Halfwidth Shift
Maximum halfwidth shift in translational search (pixels)
Envelope Masking
Enable envelope masking. Default: no
Low Pass Limit For Envelope Masking
Low pass limit for envelope mask generation (A)
Inner Mask Radius
Inner mask radius (pixels)
Inner Mask Falloff
Falloff of inner mask (pixels). Default: 10 pixels
First Iteration
Start iterating from here
Maximum Number Of Iterations
Maximum number of iterations. Default: 500
Filament Width
Filament width (A)
Center Images
Center images. Default: no
Origin Shift Multiplication Factor
Origin shift multiplication factor. Default: 1
Autoscale
Automatic down-scaling. Default: yes
Cancel
Run
